IT Project Team56 OOSD-Questify : 2025-08-14 Client Meeting notes

Created by Chang Chang on Sep 21, 2025

1. Problem Statement from client

  • Manual uploading and updating of the question bank by staff is time-consuming and inefficient.

  • There is a need for features that automate marking, enhance student engagement, and track student activity.

2. Brainstorm about Key Features & Potential AI Applications

2.1 Automated Question Generation

  • Utilize AI to generate new questions based on existing exam content.

    • Analyze previous questions to understand patterns and formats.

    • Increase question variety and create similar practice questions.

    • Ensure each generated question includes both the question and a solution (solution not shown immediately to students).

2.2 Automated Marking

  • Implement AI-driven automated marking to provide instant feedback.

    • Highlight specific areas where students make mistakes.

    • Delay revealing the full solution to encourage independent problem-solving.

2.3 Student Engagement & Motivation

  • Introduce a rewards system to keep students engaged (e.g., points, badges).

  • Implement a leaderboard to foster healthy competition.

  • Track student activity, including online time and participation.

  • Offer small prizes or recognition for top performers.

2.4 Peer-Assisted Learning

  • Encourage students to contribute their own questions and answers.

    • Award extra points for meaningful contributions.

    • Reduce staff workload by leveraging peer-generated content.

2.5 Marking Scheme & Staff Involvement

  • Develop a clear, transparent marking scheme.

  • Minimize the need for direct staff intervention by automating processes.

3. Technical Considerations

  • Programming Language: To be determined based on team expertise and project requirements.

  • Framework: Select a framework that supports rapid development and scalability.

  • API Integration: Design APIs for seamless integration with existing systems.

  • System Architecture: Define a robust architecture to support AI functionalities and user management.

  • Team Roles: Assign individual coding responsibilities to team members.
