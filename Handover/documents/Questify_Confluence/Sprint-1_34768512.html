Client Feedback Summary

  • Student identity visibility: Suggested displaying actual student names in the system to help teaching staff provide better references in the future.

  • MCQ vs Long answers: Multiple-choice questions are easy to generate with AI; long-answer questions are more difficult. Students would want to see marking rubrics.

  • Engagement concerns: Liked the idea of encouraging student contributions, but not sure how effective an AI chatbot alone will be in increasing engagement.

  • Similar tools: Asked whether platforms like Ed Discussion can provide engagement or if there are better customisable tools available.

  • LLM suggestion: Recommended looking into free large language models.

  • Admin page: It definitely would be great to have an admin page, but it is not expected to increase workload for the teaching team.

  • General feedback: Happy with the direction, but emphasized execution — encouraged us to “just do it” and move forward.

Next Steps

  • Revise user stories and motivational model based on client feedback

  • Continue planning for long-answer grading logic

  • Start planning for following sprints and codings
