IT Project Team56 OOSD-Questify : System Architecture Design Model

Created by Chang Chang, last modified by Chih-Yi Huang on Oct 30, 2025

Overview

Our system adopts a standard three-tier architecture:

  • Frontend: React, rendered in the browser.

  • Backend: Django + Django REST Framework, exposing RESTful JSON APIs.

  • Database: SQLite in development, with a migration to PostgreSQL in production.

  • External Service: OpenAI API for generating AI explanations.

  • CI/CD: GitHub + GitHub Actions for build/test automation, with deployment planned on Render.

This architecture balances simplicity for development and scalability for future use.

+ SQLite meets the needs of project, and PostgreSQL is our planned upgrade path for robustness in a deployed environment.
